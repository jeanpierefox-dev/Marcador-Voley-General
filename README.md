
# JSport Braccini

**JSport Braccini** es una plataforma integral para la gestión de torneos de voleibol, diseñada para ofrecer una experiencia profesional tanto a organizadores como a espectadores.

![JSport Banner](https://via.placeholder.com/800x200?text=JSport+Braccini)

## Funcionalidades

- **Gestión de Torneos**: Creación automática de fixtures con IA (Gemini), tablas de posiciones y gestión de equipos.
- **Live Scoring**: Panel de control en tiempo real para árbitros y entrenadores.
- **TV Overlay**: Interfaz gráfica para transmisiones en vivo (OBS/TikTok Live).
- **Estadísticas**: Seguimiento detallado de puntos, bloqueos, aces y MVPs.
- **Cloud Sync**: Sincronización entre dispositivos mediante Firebase.

## Instalación

1. Clonar el repositorio.
2. `npm install`
3. `npm run dev`

## Configuración

Configura tu API Key de Google Gemini y credenciales de Firebase en el panel de configuración de la app.
